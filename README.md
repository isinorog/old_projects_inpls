# **Data Science & Deep Learning Portfolio**

Репозиторий содержит комплекс прикладных задач по анализу данных, машинному обучению и работе с нейронными сетями. В проектах реализованы подходы от классической статистики до современных архитектур глубокого обучения.

## **Технологический стек**

Languages: Python

Libraries: PyTorch, Pandas, NumPy, Scikit-learn, Statsmodels

Models & Methods: LSTM, SARIMA, KNN, Transfer Learning, EDA
